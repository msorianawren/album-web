# TasteSkill Integration Rules

- Use TasteSkill `default` for all UI work.
- Use `soft-calm` for Oriana Wren’s luxury beige/nature portfolio style.
- Use `editorial` for About/Home/Album typography and layout.
- Use `redesign` when improving existing UI.
- Use `strict-gsap` for all GSAP animation work.
- Use `complete-output` when editing full files.
- Keep performance, accessibility, contrast, spacing, and typography checks strict.
